# End-user Manuals

Human-readable guides for non-technical users (HR managers, payroll clerks, recruiters, ops supervisors, executives) on how to use Sentinel.

## Convention

- Filename: `<role>-<topic>.md` (e.g. `hr-manager-creating-an-employee.md`, `payroll-clerk-running-a-pay-cycle.md`).
- Sections: **Who this is for**, **What you can do**, **Step-by-step**, **Common questions**, **What to do if something goes wrong**.
- Written in plain language. No jargon without a glossary.
- Linked from [`../index.md`](../index.md).

## Status

Empty. Manuals will land here as features ship.
